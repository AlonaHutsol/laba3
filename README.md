# Лабораторна робота №3

## Тема
С++. Проектування та налагодження програм оброблення масивів.

## Мета
Навчитися працювати з двовимірними масивами, знаходити елементи за заданою умовою.

## Умова завдання
Визначити мінімальний елемент головної діагоналі квадратної матриці розміром 5×5.

## Короткий опис задачі
Дано квадратну матрицю розміром 5×5. Необхідно знайти мінімальний елемент головної діагоналі.

## Опис алгоритму розв’язання
1. Задати квадратну матрицю 5×5.
2. Вивести матрицю на екран.
3. Присвоїти змінній min значення першого елемента головної діагоналі.
4. Перебрати всі елементи головної діагоналі за допомогою циклу.
5. Якщо поточний елемент менший за min, оновити значення.
6. Вивести мінімальний елемент.

## Пояснення основних етапів роботи програми
Програма працює з квадратною матрицею. Головна діагональ визначається як елементи, для яких номер рядка дорівнює номеру стовпця. Під час виконання програма проходить по цих елементах та знаходить найменше значення.

## Висновок
Було створено програму для обробки двовимірного масиву. Закріплено навички роботи з матрицями, циклами та умовними операторами.
